# Controle de estoque

Controle de estoque simples, utilizando JavaScript, HTML5 e CSS3. 
Dados salvos em localstorage com opção de backup em formato JSON.

## Funções
-Adicionar item </br>
-Saída de estoque </br>
-Entrada de estoque </br>
-Remover item </br>
-Remover todos </br>
-Backup </br>
-Restore </br>
